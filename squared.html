Home - Squared version

This version is based on straight lines and geometric shapes, trying to impact the user through the combination of shapes and branded colours, and focusing on the website logo. Background represents a physical desktop, as the basis to present the name of the tool as a relevant element, helping the user to remember it. Big buttons for tiles help to start the browsing from any kind of device, and allow the user to focus the attention on them quickly, even having an image background above.

Collapsed menu

Colllapsed menu avoids visual noise to the user, focusing on browse and search. In this case we should think on pareto principle: If 80% of the users are not using the options of the top menu, maybe we could collapse it to save space and get a more clean interface.
